# OZON Dynamic Pricing Web UI

Веб-интерфейс для системы динамического ценообразования OZON на Next.js.

## Возможности

- **Дашборд** - обзор ключевых метрик и приоритетных товаров
- **Товары** - поиск и фильтрация товаров с пагинацией
- **Аналитика** - анализ спроса, трендов и временных рядов
- **Ценообразование** - метрики для принятия решений по ценообразованию

## Установка

```bash
# Установка зависимостей
npm install

# Создание файла .env.local (если еще не создан)
echo "NEXT_PUBLIC_API_URL=http://localhost:8000" > .env.local
```

## Запуск

```bash
# Режим разработки
npm run dev

# Сборка для продакшена
npm run build
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## Настройка

Убедитесь, что:
1. Backend API запущен на http://localhost:8000
2. Переменная окружения `NEXT_PUBLIC_API_URL` указывает на правильный адрес API

## Структура проекта

```
web-ui/
├── app/              # Страницы Next.js
│   ├── page.tsx      # Главная страница (дашборд)
│   ├── products/     # Страница товаров
│   ├── analytics/    # Страница аналитики
│   └── pricing/      # Страница ценообразования
├── components/       # React компоненты
│   ├── Navbar.tsx    # Навигация
│   ├── StatsCard.tsx  # Карточка статистики
│   ├── ProductTable.tsx # Таблица товаров
│   └── ...
└── lib/              # Утилиты и API клиент
    └── api.ts        # API клиент для работы с backend
```

## Использование

1. **Дашборд** (`/`) - показывает общую статистику, топ товаров и критичные остатки
2. **Товары** (`/products`) - поиск и фильтрация товаров по категориям, брендам и другим параметрам
3. **Аналитика** (`/analytics`) - анализ спроса, трендов и временные ряды
4. **Ценообразование** (`/pricing`) - метрики для принятия решений по ценообразованию с рекомендациями
